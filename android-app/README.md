# SPM Malda Android App

This is a lightweight Android WebView app for:

https://locopilotib-gif.github.io/SPM-ANALYSIS-MALDA-DIVISION-BY-IB/

## Build APK

1. Open this `android-app` folder in Android Studio.
2. Let Android Studio sync Gradle.
3. Choose **Build > Build Bundle(s) / APK(s) > Build APK(s)**.
4. Install the APK on your Android phone.

## Included

- Loads the live SPM Analysis website.
- Supports JavaScript, local storage, file uploads, downloads, and Android back navigation.
- App name: `SPM Malda`.
- Package name: `com.imanbiswas.spmanalysis`.
